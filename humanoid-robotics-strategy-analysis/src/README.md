## Source Code Structure

This folder contains modular Wolfram Language (`.wl`) files. Each file focuses on a specific analytical question 

### File Overview
- `00_data_configuration_and_setup.wl` – Defines URLs, import helpers, and ingests the Tesla + Agility text sources.-
- `01_text_branding_wordclouds.wl` – Computes top-word counts and generates word clouds for narrative positioning.
- `02_visual_storytelling_image_tagging.wl` – Attempts image imports, uses placeholder visuals when imports fail, applies manual theme tagging, and produces visual comparisons + theme frequency charts.
- `03_visual_storytelling_analysis.wl` – Visual theme categorization of marketing imagery
- `04_leadership_background_analysis.wl` – Leadership credibility and background analysis
- `05_investment_exposure_analysis.wl` – Tesla vs HUMN ETF exposure comparison
- `06_synthesis_and_recommendation.wl` – Integrated conclusions and investment recommendation

Each file includes explanatory comments that translate analytical steps into business and strategic insight.
